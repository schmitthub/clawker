## Go Symbol Extraction Patterns

Exported identifiers in CLAUDE.md (backtick-wrapped):

```bash
grep -oE '`[A-Z][A-Za-z0-9]*`' file.md | tr -d '`' | sort -u
```

Exported symbols in Go source (excluding tests):

```bash
grep -E '^(func|type|var|const) [A-Z]' *.go | grep -v '_test.go' | sed 's/^.*:\(func\|type\|var\|const\) \([A-Za-z0-9]*\).*/\2/' | sort -u
```

For methods on types:

```bash
grep -E '^\s*func \([^)]+\) [A-Z]' *.go | grep -v '_test.go' | sed 's/.*) \([A-Za-z0-9]*\).*/\1/' | sort -u
```

## Auto Memory Audit

### Location

```
~/.claude/projects/-Users-<user>-Code-clawker/memory/
```

### MEMORY.md Index Validation

```bash
# Find memory files not referenced in MEMORY.md
for f in memory/*.md; do
  bn=$(basename "$f")
  [ "$bn" = "MEMORY.md" ] && continue
  grep -q "$bn" memory/MEMORY.md || echo "NOT INDEXED: $bn"
done

# Find broken links in MEMORY.md
grep -oE '\([^)]*\.md\)' memory/MEMORY.md | tr -d '()' | while read ref; do
  [ ! -f "memory/$ref" ] && echo "BROKEN: $ref"
done
```

## Context Budget Guidelines

| Scope | Budget | Notes |
|-------|--------|-------|
| Root CLAUDE.md | ~200 lines | Always loaded |
| Root + always-loaded rules | < 500 lines | Rules without `paths:` frontmatter |
| Total always-loaded (+ global) | < 800 lines | Includes ~/.claude/CLAUDE.md |
| Each `internal/*/CLAUDE.md` | < 200 lines | Lazy-loaded per file access |
| Each `.claude/rules/*.md` | ~50-100 lines | Path-scoped load on demand |
