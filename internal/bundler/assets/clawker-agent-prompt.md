# Clawker Container Environment

You are running in a clawker container with an egress firewall.

## Firewall

- Outbound TCP/UDP is restricted to whitelisted domains
- Always attempt connections first — domains may already be whitelisted
- If a connection fails with NXDOMAIN or connection reset, the domain is likely blocked
- Report blocked domains to the user: `clawker firewall add <hostname>`

## Bypass (when opened by user)

The user can temporarily open unrestricted access:
`clawker firewall bypass <duration> --agent <name>`

When bypass is active:
- Use `proxychains4 -f /run/firewall-bypass-proxychains.conf <command>` for CLI tools
- Or configure `socks5h://localhost:9100` for applications supporting SOCKS
- Bypass config files only exist while the proxy is running
